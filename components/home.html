Rotate

Hover me!

HTML


<div class="btn">
<p>Hover me!</p>
</div>

CSS


.btn {
/*center*/
display: flex;
justify-content: center;
align-items: center;
padding: 50px;
background: red;
transition: .8s;
}

.btn:hover {
transform: rotate(360deg);
transition: .8s;
}

Matrix

HTML


<img src="img/matrix.jpg" alt="matrix">

CSS


img {
    width: 300px;
    height: 300px;
    transition: .8s;
}

img:hover {
    transform: matrix(2, 0.7,-0.8,0.866,-50,100);
    transition: .8s;
}
/*matrix(a, b, c, d, tx, ty)
a b c d Are s describing the linear transformation.
tx ty Are s describing the translation to apply.*/
